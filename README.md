# github-action

`github action` 공부  
마! 나도 함 공부 해보자!!  
  
`github action`이 빌드하면 `server`로 신호를 줍니다.  
우와! 정말 데단해~

### 개발 환경

* NodeJS (v11.13)
* Visual Studio Code


### 어떻게 실행시킵니까?

`server`를 `npm start`하시고 `client`를 수정하여 `github action`을 작동하게 하시면 됩니다.  
Github 설정에서 `Secrets`에 `SERVER_IP`라는 이름으로 요청 주소를 입력해주시면 됩니다.  
`예) http://localhost:8000/`
